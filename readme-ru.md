# Lite Voice Terminal

Lite Voice Terminal это клиент-серверная платформа с открытым кодом для создания "легких" смарт колонок, 
не требующих использования онлайн-сервисов для распознавания голоса.

Под термином "легкий" в данном случае понимаются минимальные требования к вычислительной мощности самой колонки, 
которая может быть реализована на Raspberry Zero/ZeroW или подобном одноплатном компьютере.

Распознавание голоса при этом выполняет установленный локально сервер. Предполагается, что Intel Core i3 /16Gb 
достаточен для одновременной работы 10 терминалов.

###Возможности
* Не требует использования онлайн сервисов
* Обмен между клиентом и реализована через простой протокол поверх winsock с поддержкой SSL
* Идентификация говорящего по голосу
* 
* Возможность расширения за счет написания своих модулей 
* Взаимодействие с устройствами по MQTT протоколу (?)
* Интеграция в MajorDoMo (?)


###LVT реализован с использованием:

* [kaldi, библиотека распознавания речи](https://github.com/alphacep/kaldi)
* [vosk API, программный интерфейс для kaldi](https://github.com/alphacep/vosk-ap)
* [pymorphy2, морфологический анализатор для русского и украинского языков](https://github.com/kmike/pymorphy2)
* [RHVoice, библиотека синтеза голоса для русского языка](https://github.com/Olga-Yakovleva/RHVoice)
